## Columns {#columns}

**Volume (lots)** — traded volume in lots for the row. **Deal Count** — the
number of deals aggregated. **Contract Size** — the contract size for the symbol.
**Asset Class** — the instrument class (e.g. FX, Metals). **MT5 Node** — the MT5
server the volume came from. **Day** — the business day the row covers.
