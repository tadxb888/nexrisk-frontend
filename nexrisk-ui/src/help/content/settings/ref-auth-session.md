## What it is {#what}

**Auth & session** configures the platform's authentication and session policy —
token lifetimes, session timeout, and related security rules. **Policy preview**
shows the effect of the current settings before saving, the **Service** panel
shows auth-service health, and **Recent changes** lists recent config edits.
